# MITx Probability Capstone Recall Cards

A self-contained spaced-repetition web app containing 60 one-minute active-recall cards for MITx 6.431x, Probability: The Science of Uncertainty and Data.

## GitHub Pages

This repository is ready for branch-based GitHub Pages deployment:

1. Open **Settings → Pages**.
2. Under **Build and deployment**, select **Deploy from a branch**.
3. Select the default branch, folder **/(root)**, and save.

The site entry point is `index.html`. No build process or external dependencies are required.

## Local use

Open `index.html` directly in a modern browser.

Progress is stored locally in the browser. Use the app's Export Progress command before switching browsers or devices.
